En esta página web se podrán encontrar con definiciones base sobre la programación, algunos de los lenguajes que se trabajan en la programación y los niveles que tienen los programadores según la experiencia que adquieren.

La programación

La programación es la manipulación de datos con el fin de crear programas informáticos. Esta definición puede interpretarse de la siguiente manera, la programación no es más que darle una serie de instrucciones a la computadora en un lenguaje que esta logre entender.

Programación orientada a objetos

Esto implica desarrollar un programa como un sistema de objetos, cada uno de los cuales refleja uno u otro objeto del mundo real. Por ejemplo, si se desarrolla un programa para un taller de autos, las clases serán: auto, motor, suspensión, carrocería, sistema de frenos y otras partes de repuesto y los objetos de los modelos de la clase son los modelos específicos que se utilizan en el proceso.

Lenguajes de programación

Los lenguajes de programación es la forma de comunicarnos con la computadora, hay distintos lenguajes principalmente de bajo y de alto nivel, estos pueden ser:

html

El lenguaje html se utiliza para el desarrollo y creación de páginas web. Se componen de una serie de etiquetas que el navegador logra interpretar.

CSS

CSS es un lenguaje de hoja de estilos creado para controlar el aspecto o presentación de los documentos electrónicos definidos con html; en otras palabras, el CSS es el maquillaje para que una página web tome un buen aspecto.

Java

Java es un lenguaje de programación orientado a objetos concurrentes. Los usos más habituales del lenguaje son realmente variados. Java es muy utilizado en aplicaciones como procesadores de texto u hojas de cálculo.

Python

Este lenguaje de programación es ampliamente utilizado por empresas de todo el mundo para crear aplicaciones web, analizar datos, automatizar operaciones y crear aplicaciones empresariales fiables.

C#

Es un entorno de desarrollo de software para sistemas operativos Windows, este conjunto de herramientas se utiliza para crear sitios y aplicaciones web.

Swift

Es un lenguaje multiparadigma creado por Apple enfocado en el desarrollo de aplicaciones para IOS y macOS, está diseñado para integrarse con los framework cocoa y cocoa touch, puede usar cualquier biblioteca programada en objetive-c.

php

Es un lenguaje de código abierto que es adecuado para el desarrollo web y puede ser incrustado en html.

Java script

Es un lenguaje más conocido como scripting para páginas web, java script es un lenguaje basado en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte de programación orientada a objetos.

Niveles de programadores

Junior

Son considerados programadores junior a aquellos que cuentan con ciertas características: De 0 a 3 años de experiencia laboral, conocimientos técnicos, conocimientos básicos y proactividad.

SEMI-SENIOR

Esta es una categoría profesional intermedia cuyas características podemos definir de la siguiente manera: De 3 a 6 años de experiencia laboral, conocimientos técnicos, conocimientos básicos y proactividad media.

SENIOR

La categoría profesional más alta es la senior y cuenta con estas características: Experiencia laboral más de 6 años, conocimientos técnicos y conocimientos básicos.
